# Stock Data Downloader

Downloads historical stock market data from Yahoo Finance and saves it as a CSV file.

## Technologies
- Python
- pandas
- yfinance

## Usage

Install dependencies:
yfinance, pandas

Run the script:
main.py


The script will prompt you for a ticker, start date, and end date.
Output is saved to the `data/` folder.

## Data Source
Yahoo Finance via the `yfinance` library.
